# 📍 ランドマーク＆交差点 位置情報修正ツール

店舗情報と、それに紐づくランドマーク（目印）や最寄り交差点の座標を地図上で可視化し、OSMnxを用いて正確な位置へ修正するためのStreamlitアプリケーションです。

## 📖 概要

機械的に抽出された位置データの精度検証および修正作業を効率化します。
地図上のクリック操作で座標を修正したり、OSMnxを利用して周辺の道路ネットワーク（ノード）へ座標を「スナップ（吸着）」させることが可能です。

## ✨ 主な機能

* **可視化**: 店舗、ランドマーク、登録済み交差点を地図上に表示。
* **交差点修正**: 指定半径内の交差点ノードを検索し、ワンクリックでスナップ・更新。
* **ランドマーク編集**: 位置の微修正、削除、およびデータがない場合の新規作成。
* **進捗管理**: 「未確認」「確認済」「修正済」のステータス管理とフィルタリング。
* **自動保存**: ブラウザのリロード（F5）による作業消失を防ぐリカバリー機能。

## 🛠️ インストールと起動

### 必要要件
Python 3.9以上推奨。必要なライブラリは `requirements.txt` に記載されています。

### セットアップ
```bash
# 仮想環境の作成と有効化 (任意)
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# ライブラリのインストール
pip install -r requirements.txt
```

### アップロードするCSVファイルには以下のカラムが必要です。
| カラム名 |     必須      | 説明 |
| ------- | ----------- | ------- |
| name    | ✅ | 店舗名称 |
| lat     | ✅ | 店舗の緯度 |
|lng      | ✅ | 店舗の経度 |
|access   | - | アクセス案内文（例: 「〇〇駅 徒歩5分」） |
|landmarks_with_intersections| ✅ | ランドマーク情報のリスト（文字列またはPythonオブジェクト） |
|review_status| - | （任意）ステータス管理用。なければ自動生成されます。 |
